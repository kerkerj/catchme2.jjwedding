# catchme2 api server

This project is written in Node.js with Redis.

node version v4.4.3

```
第一次使用: p.s. 需要 node v4.4.3, 以及 redis (TODO port 設定還沒抽出來)

$ npm install

$ npm start

開發時使用:

$ npm run dev # to start nodemon service to reload files

發佈時先 build:

$ npm run build # to build ES6 js

實際執行使用:

$ npm start
```
